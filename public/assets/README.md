# Pasta de assets

Substitua estes arquivos pelos seus próprios:

- `intro.mp4` — vídeo de abertura cinematográfico
- `musica.mp3` — música de fundo romântica
- `eu.png` — bonequinho/foto do "eu"
- `cachorro.png` — bonequinho/foto do cachorro
- `foto-dela-1.png`, `foto-dela-2.png`, `foto-dela-3.png` — fotos dela (polaroid)

Enquanto os arquivos não existirem, gradientes/placeholders bonitos serão exibidos.

Para ajustar a duração da intro, altere `INTRO_DURATION_SECONDS` em
`src/components/surprise/IntroVideo.tsx`.
